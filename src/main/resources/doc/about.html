What is the OpenCV Pipeline Editor?

The OpenCV Pipeline Editor provides a graphical interface for constructing image processing workflows without writing code. It allows you to:

  • Build pipelines visually - Drag and connect processing nodes to create complex image transformations
  • See results in real-time - Monitor nodes display live output as frames flow through the pipeline
  • Process multiple sources - Work with webcams, image files, video files, or generated patterns
  • Experiment interactively - Adjust parameters with sliders and see immediate effects
  • Save and share pipelines - Export your work as JSON files for later use or sharing
  • Create nested pipelines - Encapsulate complex processing into reusable container nodes

Getting Started

  • Add nodes - Click buttons in the toolbar to add processing nodes to the canvas
  • Connect nodes - Drag from an output port (right side) to an input port (left side)
  • Configure nodes - Double-click a node to open its properties dialog
  • Run the pipeline - Click the Start button or press Ctrl+R to begin processing
  • View results - Add Monitor nodes to see the output at any point in the pipeline

Node Categories

  • Sources - Input nodes: webcam, files, generated images
  • Basic - Fundamental operations: grayscale, threshold, gain
  • Blur - Smoothing filters: Gaussian, median, bilateral
  • Edges - Edge detection: Canny, Sobel, Laplacian
  • Filter - Custom filtering and frequency domain operations
  • Morphology - Erosion, dilation, opening, closing
  • Transform - Geometric transformations: resize, crop, warp
  • Visualization - Histograms, bit planes, color analysis
  • Detection - Feature detection: contours, corners, circles
  • Dual Input - Operations requiring two inputs: blend, mask, compare
  • Nested Pipelines - Container nodes for modular pipeline design

Keyboard Shortcuts

  • Ctrl+R - Start/Stop pipeline
  • Ctrl+S - Save pipeline
  • Ctrl+O - Open pipeline
  • Ctrl+A - Select all nodes
  • Delete or Backspace - Delete selected nodes

Technology

Built with Java, JavaFX, Eclipse GEF, and OpenCV. The application uses the OpenCV computer vision library for image processing operations.
