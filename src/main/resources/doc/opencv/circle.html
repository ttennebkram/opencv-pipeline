circle

Draws a circle on an image.

Signature

Imgproc / cv2.circle(Mat img, Point center, int radius, Scalar color, int thickness)

Parameters

  • Mat img - Image where the circle is drawn (modified in place).
  • Point center - Center of the circle.
  • int radius - Radius of the circle in pixels.
  • Scalar color - Circle color in BGR format.
  • int thickness - Thickness of the circle outline. Use -1 (FILLED) for a filled circle.

Description

Examples:

// Draw circle outline (red, 2 pixels thick)
Imgproc.circle(img, new Point(100, 100), 50,
               new Scalar(0, 0, 255), 2);

// Draw filled circle (green)
Imgproc.circle(img, new Point(200, 100), 30,
               new Scalar(0, 255, 0), -1);

// Draw with anti-aliasing
Imgproc.circle(img, new Point(300, 100), 40,
               new Scalar(255, 0, 0), 2, Imgproc.LINE_AA);

Line types:

  • LINE_4 (4): 4-connected line
  • LINE_8 (8): 8-connected line (default)
  • LINE_AA (16): Anti-aliased line (smoother)
