Threshold

Applies a fixed-level threshold to each pixel. Used to create binary images or segment objects based on intensity.

Signature

Imgproc / cv2.threshold(Mat src, Mat dst, double thresh, double maxval, int type)

Parameters

  • Mat src - Input array (multiple-channel, 8-bit or 32-bit floating point).
  • Mat dst - Output array of the same size and type as src.
  • double thresh - Threshold value.
  • double maxval - Maximum value to use with THRESH_BINARY and THRESH_BINARY_INV types.
  • int type - Thresholding type (see table below).

Threshold Types

TypeValueOperation
THRESH_BINARY0dst = (src > thresh) ? maxval : 0
THRESH_BINARY_INV1dst = (src > thresh) ? 0 : maxval
THRESH_TRUNC2dst = (src > thresh) ? thresh : src
THRESH_TOZERO3dst = (src > thresh) ? src : 0
THRESH_TOZERO_INV4dst = (src > thresh) ? 0 : src

Special Flags

These can be combined with a threshold type using the OR operator:

  • THRESH_OTSU - Automatically determine optimal threshold using Otsu's method (8-bit images only)
  • THRESH_TRIANGLE - Automatically determine threshold using the triangle algorithm

Description

The threshold function is fundamental for creating binary images. It's commonly used for:

  • Separating foreground from background
  • Creating masks for further processing
  • Object detection preprocessing
  • Document binarization

See Also

  • inRange() - Color range thresholding
