cornerHarris

Harris corner detector. Finds corners based on the local gradient structure of the image.

Signature

Imgproc / cv2.cornerHarris(Mat src, Mat dst, int blockSize, int ksize, double k)

Parameters

  • Mat src - Input single-channel 8-bit or floating-point image.
  • Mat dst - Image to store the Harris detector responses. Type CV_32FC1, same size as src.
  • int blockSize - Neighborhood size for corner detection (typically 2-5).
  • int ksize - Aperture parameter for Sobel operator (typically 3).
  • double k - Harris detector free parameter (typically 0.04-0.06).

Description

The Harris corner detector uses the gradient covariance matrix to detect corners. It computes a response R for each pixel:

R = det(M) - k * trace(M)Â²

Where M is the structure tensor (sum of gradient products over the neighborhood).

Interpreting the output:

  • Large positive R: Corner
  • Negative R: Edge
  • Small |R|: Flat region

Typical workflow:

// Detect corners
Mat harris = new Mat();
Imgproc.cornerHarris(gray, harris, 2, 3, 0.04);

// Normalize for visualization
Core.normalize(harris, harris, 0, 255, Core.NORM_MINMAX);

// Threshold to find strong corners
Mat corners = new Mat();
Core.compare(harris, new Scalar(threshold), corners, Core.CMP_GT);
