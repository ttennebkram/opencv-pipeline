SimpleBlobDetector

Detects blobs (regions) in an image based on various criteria like color, area, circularity, convexity, and inertia ratio.

Signature

Features2d / cv2.SimpleBlobDetector_create(SimpleBlobDetector_Params params)

Description

The blob detector works by:

  1. Thresholding the image at multiple levels
  2. Finding connected components (blobs) at each level
  3. Filtering blobs by specified criteria
  4. Merging nearby blobs across threshold levels

Filter Parameters

FilterDescription
filterByColorFilter by blob color (dark vs light)
filterByAreaFilter by blob size (minArea, maxArea)
filterByCircularityFilter by how circular (4πArea/perimeter²)
filterByConvexityFilter by convexity (Area/ConvexHullArea)
filterByInertiaFilter by elongation (circle=1, line=0)

Usage:

// Create detector with default parameters
SimpleBlobDetector detector = SimpleBlobDetector.create();

// Detect blobs
MatOfKeyPoint keypoints = new MatOfKeyPoint();
detector.detect(gray, keypoints);

// Draw detected blobs
Mat output = new Mat();
Features2d.drawKeypoints(image, keypoints, output, new Scalar(0, 0, 255),
    Features2d.DrawMatchesFlags_DRAW_RICH_KEYPOINTS);

Custom parameters:

SimpleBlobDetector_Params params = new SimpleBlobDetector_Params();
params.set_filterByArea(true);
params.set_minArea(100);
params.set_maxArea(5000);
params.set_filterByCircularity(true);
params.set_minCircularity(0.8f);  // Detect round blobs
SimpleBlobDetector detector = SimpleBlobDetector.create(params);
