Box Blur (blur)

Blurs an image using a normalized box filter. This is the simplest and fastest blur operation.

Signature

Imgproc / cv2.blur(Mat src, Mat dst, Size ksize)

Parameters

  • Mat src - Input image; it can have any number of channels, which are processed independently.
  • Mat dst - Output image of the same size and type as src.
  • Size ksize - Blurring kernel size (width, height). Both values must be positive and odd.

Description

The box blur (also called box filter or averaging filter) replaces each pixel with the average of all pixels in the kernel area. It uses a normalized kernel:

K = 1/(ksize.width * ksize.height) * [ 1 1 1 ... ]
                                       [ 1 1 1 ... ]
                                       [ ... ... ]

This is equivalent to convolving the image with a kernel filled with ones and then dividing by the kernel area.

Characteristics:

  • Fastest blur operation
  • Equal weighting for all pixels in the kernel
  • Can produce "boxy" artifacts on edges
  • Good for quick noise reduction

See Also

  • GaussianBlur() - Weighted blur (better quality)
  • medianBlur() - Good for salt-and-pepper noise
  • bilateralFilter() - Edge-preserving blur
