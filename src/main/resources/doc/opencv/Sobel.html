Sobel Operator

Calculates the first, second, third, or mixed image derivatives using an extended Sobel operator.

Signature

Imgproc / cv2.Sobel(Mat src, Mat dst, int ddepth, int dx, int dy, int ksize)

Parameters

  • Mat src - Input image.
  • Mat dst - Output image of the same size and same number of channels as src.
  • int ddepth - Output image depth. Use -1 for same as source, or CV_16S, CV_32F, CV_64F to avoid overflow.
  • int dx - Order of the derivative in x direction (0, 1, 2, or 3).
  • int dy - Order of the derivative in y direction (0, 1, 2, or 3).
  • int ksize - Size of the extended Sobel kernel; must be 1, 3, 5, or 7. Default is 3.

Description

The Sobel operator computes the gradient of the image intensity. It combines Gaussian smoothing and differentiation, making it more resistant to noise than simple gradient operators.

Common usage patterns:

  • dx=1, dy=0: Horizontal edges (gradient in X direction)
  • dx=0, dy=1: Vertical edges (gradient in Y direction)
  • dx=1, dy=1: Diagonal edges
  • dx=2, dy=0: Second derivative in X (edge curvature)

3x3 Sobel kernels:

Horizontal (dx=1):    Vertical (dy=1):
[-1  0  1]            [-1 -2 -1]
[-2  0  2]            [ 0  0  0]
[-1  0  1]            [ 1  2  1]

Depth recommendation: Use CV_16S or CV_32F for ddepth to capture negative values and avoid overflow. Convert to CV_8U for display using convertScaleAbs().

See Also

  • Laplacian() - Second-order derivative
