ORB Feature Detector

Oriented FAST and Rotated BRIEF. A fast, efficient feature detector and descriptor that is free to use (no patents). Good for real-time applications.

Signature

Features2d / cv2.ORB_create(int nfeatures)

Parameters

  • int nfeatures - Maximum number of features to detect (default 500).

Description

ORB combines:

  • FAST keypoint detector (modified for orientation)
  • BRIEF descriptor (rotated for orientation invariance)

Key properties:

  • Very fast (good for real-time)
  • Rotation invariant
  • Resistant to noise
  • Binary descriptor (fast matching with Hamming distance)
  • Free to use (unlike SIFT/SURF patents)

Usage:

// Create detector
ORB orb = ORB.create(500);

// Detect keypoints and compute descriptors
MatOfKeyPoint keypoints = new MatOfKeyPoint();
Mat descriptors = new Mat();
orb.detectAndCompute(gray, new Mat(), keypoints, descriptors);

// Draw keypoints
Mat output = new Mat();
Features2d.drawKeypoints(image, keypoints, output);

Feature matching:

// Match features between two images
BFMatcher matcher = BFMatcher.create(Core.NORM_HAMMING, true);
MatOfDMatch matches = new MatOfDMatch();
matcher.match(descriptors1, descriptors2, matches);
