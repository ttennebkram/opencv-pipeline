inRange - Color Range

Checks if array elements lie between the elements of two other arrays. Creates a binary mask of pixels within the specified color range.

Signature

Core / cv2.inRange(Mat src, Scalar lowerb, Scalar upperb, Mat dst)

Parameters

  • Mat src - Input array (single or multi-channel).
  • Scalar lowerb - Inclusive lower boundary (minimum values for each channel).
  • Scalar upperb - Inclusive upper boundary (maximum values for each channel).
  • Mat dst - Output array of the same size as src and CV_8U type (binary mask).

Description

For each pixel, the function checks:

dst(x,y) = 255 if lowerb ≤ src(x,y) ≤ upperb (all channels)
         = 0   otherwise

The output is a binary mask where white pixels (255) indicate pixels within the range.

Tip: For color detection, convert to HSV first. The Hue channel is more robust to lighting changes than BGR values.

Note: OpenCV uses H range 0-179, not 0-360. Multiply standard hue values by 0.5.

Example Usage

// Detect blue objects
Mat hsv = new Mat();
Imgproc.cvtColor(src, hsv, Imgproc.COLOR_BGR2HSV);

Scalar lowerBlue = new Scalar(100, 50, 50);
Scalar upperBlue = new Scalar(130, 255, 255);

Mat mask = new Mat();
Core.inRange(hsv, lowerBlue, upperBlue, mask);
