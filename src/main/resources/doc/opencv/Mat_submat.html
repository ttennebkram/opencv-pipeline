Mat.submat - Crop / Region of Interest

Extracts a rectangular sub-region (Region of Interest) from a Mat. The result shares data with the original image.

Signature

Mat submat = mat.submat(Rect roi)

Mat submat = mat.submat(int rowStart, int rowEnd, int colStart, int colEnd)

Mat submat = new Mat(mat, Rect roi)

Parameters

  • Rect roi - Rectangle defining the region to extract.
  • int rowStart, rowEnd - Row range (exclusive end).
  • int colStart, colEnd - Column range (exclusive end).

Description

Important: submat() creates a view into the original data. Changes to the submat affect the original image, and vice versa. Use .clone() if you need an independent copy.

Examples:

// Extract region using Rect
Rect roi = new Rect(100, 50, 200, 150);  // x, y, width, height
Mat cropped = img.submat(roi);

// Extract using row/column ranges
Mat cropped = img.submat(50, 200, 100, 300);  // rows 50-199, cols 100-299

// Using Mat constructor
Mat cropped = new Mat(img, new Rect(0, 0, 100, 100));

// Get independent copy (doesn't share data)
Mat croppedCopy = img.submat(roi).clone();

// Modify a region of the original
Mat region = img.submat(new Rect(10, 10, 50, 50));
region.setTo(new Scalar(0, 0, 255));  // Fill region with red

Common use cases:

  • Cropping images to focus on regions of interest
  • Processing only a portion of a large image
  • Extracting detected objects for further analysis
  • Efficiently modifying image regions without copying
