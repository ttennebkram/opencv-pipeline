cvtColor - Color Conversion

Converts an image from one color space to another. This is one of the most frequently used OpenCV functions.

Signature

Imgproc / cv2.cvtColor(Mat src, Mat dst, int code)

Parameters

  • Mat src - Input image: 8-bit unsigned, 16-bit unsigned, or single-precision floating-point.
  • Mat dst - Output image of the same size and depth as src.
  • int code - Color space conversion code (see table below).

Common Conversion Codes

CodeDescription
COLOR_BGR2GRAYBGR to Grayscale
COLOR_GRAY2BGRGrayscale to BGR (3-channel)
COLOR_BGR2RGBBGR to RGB (swap channels)
COLOR_BGR2HSVBGR to HSV (Hue, Saturation, Value)
COLOR_HSV2BGRHSV to BGR
COLOR_BGR2LabBGR to CIE Lab
COLOR_BGR2YCrCbBGR to YCrCb (luma, chroma)
COLOR_BGR2HLSBGR to HLS (Hue, Lightness, Saturation)

Description

Grayscale conversion (COLOR_BGR2GRAY):

Gray = 0.299*R + 0.587*G + 0.114*B

This weighted sum reflects human perception of brightness.

HSV color space:

  • H (Hue): 0-179 in OpenCV (color type: red, green, blue, etc.)
  • S (Saturation): 0-255 (color intensity)
  • V (Value): 0-255 (brightness)

HSV is useful for color-based object detection since H is independent of lighting.

Important: OpenCV uses BGR order by default, not RGB. When loading images from other sources or displaying with other libraries, you may need COLOR_BGR2RGB.
