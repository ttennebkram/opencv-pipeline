add

Calculates the per-element sum of two arrays or an array and a scalar. Values are saturated (clamped) to the valid range.

Signature

Core / cv2.add(Mat src1, Mat src2, Mat dst, Mat mask, int dtype)

Parameters

  • Mat src1 - First input array or scalar.
  • Mat src2 - Second input array or scalar.
  • Mat dst - Output array with the same size and number of channels as the inputs.
  • Mat mask - Optional operation mask (8-bit single channel). Where mask is 0, dst is unchanged.
  • int dtype - Optional depth of the output array. Default is -1 (same as inputs).

Description

Computes element-wise addition with saturation:

dst(x,y) = saturate(src1(x,y) + src2(x,y))
Saturation: For 8-bit images, results are clamped to 0-255. Values exceeding 255 become 255 (no overflow/wrap-around).

Common use cases:

  • Brighten images: Add a scalar value to increase brightness
  • Combine images: Layer two images together
  • Add overlays: Combine image with graphics/text

Example:

// Brighten an image by 50
Mat brighter = new Mat();
Core.add(image, new Scalar(50, 50, 50), brighter);

// Combine two images
Mat combined = new Mat();
Core.add(image1, image2, combined);

  • addWeighted() - Weighted sum (blending)
