calcHist - Calculate Histogram

Calculates the histogram of an image. A histogram shows the distribution of pixel intensities.

Signature

Imgproc / cv2.calcHist(List<Mat> images, MatOfInt channels, Mat mask, Mat hist, MatOfInt histSize, MatOfFloat ranges)

Parameters

  • List<Mat> images - Source images (typically just one).
  • MatOfInt channels - Channel indices to compute histogram for (e.g., 0 for grayscale).
  • Mat mask - Optional mask (only pixels where mask != 0 are counted). Use empty Mat for no mask.
  • Mat hist - Output histogram.
  • MatOfInt histSize - Number of bins for each dimension (typically 256 for full range).
  • MatOfFloat ranges - Value ranges (typically 0-256 for 8-bit images).

Description

A histogram counts how many pixels have each intensity value. It's fundamental for:

  • Image analysis and statistics
  • Contrast enhancement (histogram equalization)
  • Thresholding decisions
  • Image comparison

Example - Grayscale histogram:

List<Mat> images = Arrays.asList(gray);
MatOfInt channels = new MatOfInt(0);
Mat hist = new Mat();
MatOfInt histSize = new MatOfInt(256);
MatOfFloat ranges = new MatOfFloat(0, 256);

Imgproc.calcHist(images, channels, new Mat(), hist, histSize, ranges);

// Draw histogram
int histH = 300, histW = 512;
Mat histImage = new Mat(histH, histW, CvType.CV_8UC3, new Scalar(255, 255, 255));
Core.normalize(hist, hist, 0, histH, Core.NORM_MINMAX);

for (int i = 1; i < 256; i++) {
    Imgproc.line(histImage,
        new Point((i-1) * 2, histH - hist.get(i-1, 0)[0]),
        new Point(i * 2, histH - hist.get(i, 0)[0]),
        new Scalar(0, 0, 0), 2);
}
