putText

Draws a text string on an image.

Signature

Imgproc / cv2.putText(Mat img, String text, Point org, int fontFace, double fontScale, Scalar color, int thickness)

Parameters

  • Mat img - Image where the text is drawn.
  • String text - Text string to be drawn.
  • Point org - Bottom-left corner of the text string in the image.
  • int fontFace - Font type (see table below).
  • double fontScale - Font scale factor (multiplies the font-specific base size).
  • Scalar color - Text color in BGR format.
  • int thickness - Thickness of the lines used to draw text.

Font Faces

ConstantDescription
FONT_HERSHEY_SIMPLEXNormal size sans-serif
FONT_HERSHEY_PLAINSmall size sans-serif
FONT_HERSHEY_DUPLEXNormal size sans-serif, more complex
FONT_HERSHEY_COMPLEXNormal size serif
FONT_HERSHEY_TRIPLEXNormal size serif, more complex
FONT_HERSHEY_SCRIPT_SIMPLEXHand-writing style
FONT_HERSHEY_SCRIPT_COMPLEXHand-writing style, more complex

Add FONT_ITALIC to any font for italic text.

Description

Examples:

// Simple text
Imgproc.putText(img, "Hello OpenCV",
    new Point(50, 50),
    Imgproc.FONT_HERSHEY_SIMPLEX, 1.0,
    new Scalar(0, 255, 0), 2);

// Italic text
Imgproc.putText(img, "Italic Text",
    new Point(50, 100),
    Imgproc.FONT_HERSHEY_SIMPLEX | Imgproc.FONT_ITALIC, 1.0,
    new Scalar(255, 0, 0), 2);

// Get text size for centering
Size textSize = Imgproc.getTextSize("Centered",
    Imgproc.FONT_HERSHEY_SIMPLEX, 1.0, 2, null);
int x = (img.cols() - (int)textSize.width) / 2;
Imgproc.putText(img, "Centered", new Point(x, 200),
    Imgproc.FONT_HERSHEY_SIMPLEX, 1.0, new Scalar(0, 0, 255), 2);
