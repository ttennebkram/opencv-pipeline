Scharr Operator

Calculates the first x- or y- image derivative using the Scharr operator. More accurate than the standard Sobel for 3x3 kernels.

Signature

Imgproc / cv2.Scharr(Mat src, Mat dst, int ddepth, int dx, int dy)

Parameters

  • Mat src - Input image.
  • Mat dst - Output image of the same size and same number of channels as src.
  • int ddepth - Output image depth. Use -1 for same as source, or CV_16S, CV_32F to avoid overflow.
  • int dx - Order of the derivative x (0 or 1).
  • int dy - Order of the derivative y (0 or 1).

Description

The Scharr operator is equivalent to Sobel with ksize=FILTER_SCHARR (-1). It provides better rotational symmetry and more accurate gradient estimation for small kernels.

Scharr kernels:

Horizontal (dx=1):    Vertical (dy=1):
[-3   0   3]          [-3  -10  -3]
[-10  0  10]          [ 0    0   0]
[-3   0   3]          [ 3   10   3]

Advantages over Sobel 3x3:

  • Better rotational symmetry
  • More accurate gradient direction
  • Less sensitivity to diagonal artifacts
  • Preferred for precise edge detection

Note: Unlike Sobel, Scharr only supports first-order derivatives (dx or dy = 0 or 1, and dx + dy = 1). One of dx or dy must be 1 and the other must be 0.
