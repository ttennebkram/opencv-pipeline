bitwise_not

Inverts every bit of an array. Also known as complement or inversion.

Signature

Core / cv2.bitwise_not(Mat src, Mat dst, Mat mask)

Parameters

  • Mat src - Input array.
  • Mat dst - Output array with the same size and type as src.
  • Mat mask - Optional operation mask, 8-bit single channel (optional, can be empty Mat).

Description

Computes bitwise NOT (inversion) for each element:

dst(x,y) = NOT src(x,y) = 255 - src(x,y)    for 8-bit images

Effects:

  • Black (0) becomes white (255)
  • White (255) becomes black (0)
  • All values are inverted: new_value = 255 - old_value

Common use cases:

  • Invert masks: Swap foreground/background in a binary mask
  • Create negative images: Photo negative effect
  • Invert thresholds: Alternative to THRESH_BINARY_INV

Example:

// Invert a binary mask to swap foreground/background
Mat invertedMask = new Mat();
Core.bitwise_not(mask, invertedMask);

// Create a photo negative
Mat negative = new Mat();
Core.bitwise_not(colorImage, negative);
