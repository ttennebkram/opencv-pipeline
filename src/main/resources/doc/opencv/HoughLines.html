HoughLines

Finds lines in a binary image using the Standard Hough Transform. Returns lines as (rho, theta) pairs representing infinite lines.

Signature

Imgproc / cv2.HoughLines(Mat image, Mat lines, double rho, double theta, int threshold)

Parameters

  • Mat image - 8-bit, single-channel binary source image (typically from Canny edge detection).
  • Mat lines - Output vector of lines. Each line is represented by a 2-element vector (ρ, θ).
  • double rho - Distance resolution of the accumulator in pixels (typically 1).
  • double theta - Angle resolution of the accumulator in radians (typically π/180).
  • int threshold - Accumulator threshold. Only lines with enough votes are returned.

Description

Lines are represented in polar form:

ρ = x·cos(θ) + y·sin(θ)

Where ρ is the distance from origin to the line, and θ is the angle of the perpendicular.

Typical workflow:

  1. Convert to grayscale
  2. Apply edge detection (Canny)
  3. Run HoughLines
  4. Convert (ρ, θ) to line endpoints for drawing

Drawing detected lines:

for (int i = 0; i < lines.rows(); i++) {
    double rho = lines.get(i, 0)[0];
    double theta = lines.get(i, 0)[1];
    double a = Math.cos(theta), b = Math.sin(theta);
    double x0 = a * rho, y0 = b * rho;
    Point pt1 = new Point(x0 + 1000*(-b), y0 + 1000*(a));
    Point pt2 = new Point(x0 - 1000*(-b), y0 - 1000*(a));
    Imgproc.line(dst, pt1, pt2, new Scalar(0, 0, 255), 2);
}

Note: For line segments with endpoints, use HoughLinesP (Probabilistic Hough Transform) instead.
