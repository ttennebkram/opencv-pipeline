Dilate

Dilates an image using a structuring element. Expands bright regions (or white areas in binary images).

Signature

Imgproc / cv2.dilate(Mat src, Mat dst, Mat kernel)

Parameters

  • Mat src - Input image; can have any number of channels, processed independently.
  • Mat dst - Output image of the same size and type as src.
  • Mat kernel - Structuring element. Use getStructuringElement() to create, or pass empty Mat for 3x3 rectangle.

Description

Dilation replaces each pixel with the local maximum over the area of the structuring element:

dst(x,y) = max(src(x+x', y+y')) for (x',y') in kernel

Effects on binary images:

  • White regions grow/expand
  • Black regions shrink
  • Small holes are filled
  • Objects connect if close together

Effects on grayscale images:

  • Bright areas expand
  • Dark details are reduced
  • Overall image appears brighter

Common structuring elements:

// 5x5 rectangular kernel
Mat kernel = Imgproc.getStructuringElement(Imgproc.MORPH_RECT, new Size(5, 5));

// 5x5 elliptical kernel
Mat kernel = Imgproc.getStructuringElement(Imgproc.MORPH_ELLIPSE, new Size(5, 5));

// 5x5 cross-shaped kernel
Mat kernel = Imgproc.getStructuringElement(Imgproc.MORPH_CROSS, new Size(5, 5));

See Also

  • erode() - Opposite operation (shrink bright regions)
  • morphologyEx() - Advanced morphological operations
