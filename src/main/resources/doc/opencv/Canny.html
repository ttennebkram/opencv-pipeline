Canny Edge Detection

Finds edges in an image using the Canny algorithm.

Signature

Imgproc / cv2.Canny(Mat image, Mat edges, double threshold1, double threshold2)

Parameters

  • Mat image - 8-bit input image.
  • Mat edges - Output edge map; single channels 8-bit image, which has the same size as image.
  • double threshold1 - First threshold for the hysteresis procedure.
  • double threshold2 - Second threshold for the hysteresis procedure.

Description

The Canny edge detector is a multi-stage algorithm:

  1. Noise Reduction: Apply Gaussian filter to smooth the image
  2. Gradient Calculation: Find intensity gradients using Sobel operator
  3. Non-maximum Suppression: Thin the edges to 1-pixel width
  4. Hysteresis Thresholding: Use two thresholds to determine final edges

Edges with intensity gradient above threshold2 are considered strong edges. Edges between threshold1 and threshold2 are kept only if connected to strong edges.
