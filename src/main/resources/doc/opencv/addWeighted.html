addWeighted

Calculates the weighted sum of two arrays. Used for blending, cross-fading, and alpha compositing.

Signature

Core / cv2.addWeighted(Mat src1, double alpha, Mat src2, double beta, double gamma, Mat dst)

Parameters

  • Mat src1 - First input array.
  • double alpha - Weight of the first array (0.0 to 1.0 typically).
  • Mat src2 - Second input array of the same size and type as src1.
  • double beta - Weight of the second array (0.0 to 1.0 typically).
  • double gamma - Scalar added to each sum (brightness adjustment).
  • Mat dst - Output array with the same size and type as inputs.

Description

Computes the weighted sum:

dst(x,y) = saturate(alpha * src1(x,y) + beta * src2(x,y) + gamma)

For standard blending, use alpha + beta = 1.0:

  • alpha=0.7, beta=0.3: 70% of first image, 30% of second
  • alpha=0.5, beta=0.5: Equal blend of both images
  • alpha=1.0, beta=0.0: Only first image

Common use cases:

  • Image blending: Combine two images with transparency
  • Cross-fade: Transition between images/video frames
  • Overlay effects: Add graphics with semi-transparency
  • Image enhancement: Combine original with filtered version

Example:

// Blend two images 50-50
Mat blended = new Mat();
Core.addWeighted(image1, 0.5, image2, 0.5, 0, blended);

// Sharpen: add weighted difference between original and blurred
Mat blurred = new Mat();
Imgproc.GaussianBlur(image, blurred, new Size(0, 0), 3);
Mat sharpened = new Mat();
Core.addWeighted(image, 1.5, blurred, -0.5, 0, sharpened);
