ellipse

Draws an ellipse outline, filled ellipse, or elliptic arc.

Signature

Imgproc / cv2.ellipse(Mat img, Point center, Size axes, double angle, double startAngle, double endAngle, Scalar color, int thickness)

Parameters

  • Mat img - Image where the ellipse is drawn.
  • Point center - Center of the ellipse.
  • Size axes - Half of the size of the ellipse main axes (width/2, height/2).
  • double angle - Ellipse rotation angle in degrees.
  • double startAngle - Starting angle of the elliptic arc in degrees.
  • double endAngle - Ending angle of the elliptic arc in degrees.
  • Scalar color - Ellipse color in BGR.
  • int thickness - Line thickness. -1 for filled.

Description

Examples:

// Draw full ellipse (100x50 pixels, no rotation)
Imgproc.ellipse(img, new Point(200, 150),
    new Size(100, 50),  // axes (half-width, half-height)
    0,                   // no rotation
    0, 360,              // full ellipse
    new Scalar(0, 255, 0), 2);

// Draw rotated ellipse (45 degrees)
Imgproc.ellipse(img, new Point(400, 150),
    new Size(80, 40), 45, 0, 360,
    new Scalar(255, 0, 0), 2);

// Draw arc (quarter ellipse)
Imgproc.ellipse(img, new Point(200, 300),
    new Size(60, 60), 0, 0, 90,  // 0 to 90 degrees
    new Scalar(0, 0, 255), 3);

// Draw filled ellipse
Imgproc.ellipse(img, new Point(400, 300),
    new Size(50, 30), 0, 0, 360,
    new Scalar(255, 255, 0), -1);
