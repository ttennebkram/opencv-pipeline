resize

Resizes an image to the specified dimensions using various interpolation methods.

Signature

Imgproc.resize(Mat src, Mat dst, Size dsize, double fx, double fy, int interpolation)

Parameters

  • Mat src - Input image.
  • Mat dst - Output image; it has the size dsize or the size computed from src.size(), fx, and fy.
  • Size dsize - Output image size. If it equals zero, it is computed as: dsize = Size(round(fx*src.cols), round(fy*src.rows)).
  • double fx - Scale factor along the horizontal axis. If 0, computed from dsize.
  • double fy - Scale factor along the vertical axis. If 0, computed from dsize.
  • int interpolation - Interpolation method (see below).

Interpolation Methods

  • INTER_NEAREST - Nearest-neighbor interpolation (fastest, pixelated)
  • INTER_LINEAR - Bilinear interpolation (good balance, default)
  • INTER_AREA - Resampling using pixel area relation (best for shrinking)
  • INTER_CUBIC - Bicubic interpolation over 4x4 neighborhood (smoother)
  • INTER_LANCZOS4 - Lanczos interpolation over 8x8 neighborhood (high quality)

Description

The function resize resizes the image src down to or up to the specified size. When shrinking an image, INTER_AREA is generally best. When enlarging, INTER_CUBIC or INTER_LINEAR are recommended.

  • Mat.submat() - Crop a region of interest
