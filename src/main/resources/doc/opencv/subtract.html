subtract

Calculates the per-element difference of two arrays or an array and a scalar. Values are saturated (clamped) to the valid range.

Signature

Core / cv2.subtract(Mat src1, Mat src2, Mat dst, Mat mask, int dtype)

Parameters

  • Mat src1 - First input array or scalar.
  • Mat src2 - Second input array or scalar (subtracted from src1).
  • Mat dst - Output array with the same size and number of channels as the inputs.
  • Mat mask - Optional operation mask (8-bit single channel). Where mask is 0, dst is unchanged.
  • int dtype - Optional depth of the output array. Default is -1 (same as inputs).

Description

Computes element-wise subtraction with saturation:

dst(x,y) = saturate(src1(x,y) - src2(x,y))
Saturation: For 8-bit images, negative results become 0 (no underflow/wrap-around). Use absdiff() if you want absolute differences.

Common use cases:

  • Darken images: Subtract a scalar to decrease brightness
  • Background subtraction: Remove static background from video
  • Change detection: Find differences between frames

Example:

// Darken an image by 30
Mat darker = new Mat();
Core.subtract(image, new Scalar(30, 30, 30), darker);

// Simple background subtraction
Mat foreground = new Mat();
Core.subtract(currentFrame, backgroundFrame, foreground);
