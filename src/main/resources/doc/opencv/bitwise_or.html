bitwise_or

Calculates the per-element bit-wise disjunction (OR) of two arrays or an array and a scalar.

Signature

Core / cv2.bitwise_or(Mat src1, Mat src2, Mat dst, Mat mask)

Parameters

  • Mat src1 - First input array or scalar.
  • Mat src2 - Second input array or scalar.
  • Mat dst - Output array with the same size and type as the inputs.
  • Mat mask - Optional operation mask, 8-bit single channel (optional, can be empty Mat).

Description

Computes bitwise OR for each element:

dst(x,y) = src1(x,y) OR src2(x,y)    if mask(x,y) != 0

Common use cases:

  • Combining masks: Union of two binary masks
  • Merging regions: Combine detected regions from different methods
  • Adding overlays: Combine image layers

Example - combining masks:

// Combine two detection masks (e.g., detect red OR blue objects)
Mat redMask = new Mat();
Mat blueMask = new Mat();
Core.inRange(hsv, redLower, redUpper, redMask);
Core.inRange(hsv, blueLower, blueUpper, blueMask);

Mat combinedMask = new Mat();
Core.bitwise_or(redMask, blueMask, combinedMask);
