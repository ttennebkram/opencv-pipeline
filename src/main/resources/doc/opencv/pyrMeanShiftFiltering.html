pyrMeanShiftFiltering

Performs mean-shift filtering using a color/spatial pyramid. Creates a smoothed, posterized look while preserving edges.

Signature

Imgproc / cv2.pyrMeanShiftFiltering(Mat src, Mat dst, double sp, double sr)

Parameters

  • Mat src - Input 8-bit 3-channel image.
  • Mat dst - Output image of the same size and type as src.
  • double sp - Spatial window radius (affects smoothness).
  • double sr - Color window radius (affects color flattening).

Description

Mean-shift filtering groups pixels with similar colors that are spatially close, creating regions of uniform color while preserving edges.

Performance: This operation is relatively slow. For large images, consider resizing first or using a smaller sp value.

Parameter effects:

  • sp (spatial): Larger values = more blurring, smoother boundaries
  • sr (color): Larger values = more color flattening, fewer distinct colors

Example:

// Basic mean-shift filtering
Mat result = new Mat();
Imgproc.pyrMeanShiftFiltering(src, result, 20, 40);

// Create cartoon effect
Mat filtered = new Mat();
Imgproc.pyrMeanShiftFiltering(src, filtered, 15, 50);

// Add edges for cartoon look
Mat gray = new Mat();
Imgproc.cvtColor(src, gray, Imgproc.COLOR_BGR2GRAY);
Mat edges = new Mat();
Imgproc.adaptiveThreshold(gray, edges, 255,
    Imgproc.ADAPTIVE_THRESH_MEAN_C, Imgproc.THRESH_BINARY, 9, 2);
Imgproc.cvtColor(edges, edges, Imgproc.COLOR_GRAY2BGR);
Core.bitwise_and(filtered, edges, cartoon);

Use cases:

  • Image segmentation preprocessing
  • Creating artistic/painted effects
  • Noise reduction with edge preservation
  • Cartoon/posterization effects
