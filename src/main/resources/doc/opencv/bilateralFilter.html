Bilateral Filter

Applies a bilateral filter to an image. This filter smooths images while preserving edges by considering both spatial distance and intensity difference.

Signature

Imgproc / cv2.bilateralFilter(Mat src, Mat dst, int d, double sigmaColor, double sigmaSpace)

Parameters

  • Mat src - Source 8-bit or floating-point, 1-channel or 3-channel image.
  • Mat dst - Destination image of the same size and type as src.
  • int d - Diameter of each pixel neighborhood. If non-positive, it is computed from sigmaSpace. A value of 5-9 is typical.
  • double sigmaColor - Filter sigma in the color space. Larger value means farther colors will be mixed together. Values in the range 50-150 are typical.
  • double sigmaSpace - Filter sigma in the coordinate space. Larger value means farther pixels will influence each other. Values in the range 50-150 are typical.

Description

The bilateral filter is an edge-preserving smoothing filter. Unlike Gaussian blur which only considers spatial distance, the bilateral filter also considers color/intensity similarity.

How it works:

  • Spatial weight: Pixels closer to the center have more influence (like Gaussian blur)
  • Range weight: Pixels with similar intensity/color have more influence
  • The combination preserves edges while smoothing flat regions
Performance Note: Bilateral filtering is significantly slower than other blur operations. For real-time applications, consider using a smaller diameter or fewer iterations.

Typical parameter values:

  • d = 9: Good balance of quality and speed
  • sigmaColor = 75: Moderate color sensitivity
  • sigmaSpace = 75: Moderate spatial extent

Use cases:

  • Noise reduction while keeping edges sharp
  • Skin smoothing in portrait photos
  • Pre-processing for edge detection
  • Creating a "cartoon" or painted effect

See Also

  • GaussianBlur() - Faster but doesn't preserve edges
  • medianBlur() - Good for salt-and-pepper noise
  • blur() - Simple box filter
