arrowedLine

Draws a line segment with an arrow tip pointing from the first point to the second point.

Signature

Imgproc / cv2.arrowedLine(Mat img, Point pt1, Point pt2, Scalar color, int thickness, int lineType, int shift, double tipLength)

Parameters

  • Mat img - Image where the arrow is drawn.
  • Point pt1 - Starting point of the arrow (tail).
  • Point pt2 - Ending point of the arrow (tip).
  • Scalar color - Arrow color in BGR format.
  • int thickness - Line thickness in pixels.
  • int lineType - Type of line (LINE_4, LINE_8, LINE_AA).
  • int shift - Number of fractional bits in point coordinates (usually 0).
  • double tipLength - Length of the arrow tip relative to the arrow length (default 0.1 = 10%).

Description

Draws an arrow from pt1 to pt2. The arrow head is drawn at pt2.

Examples:

// Simple arrow
Imgproc.arrowedLine(img, new Point(50, 50), new Point(200, 100),
                    new Scalar(0, 0, 255), 2);

// Arrow with larger tip (20% of line length)
Imgproc.arrowedLine(img, new Point(50, 100), new Point(200, 150),
                    new Scalar(0, 255, 0), 2, Imgproc.LINE_AA, 0, 0.2);

// Thick arrow with anti-aliasing
Imgproc.arrowedLine(img, new Point(50, 150), new Point(200, 200),
                    new Scalar(255, 0, 0), 3, Imgproc.LINE_AA, 0, 0.15);

Use cases:

  • Visualizing optical flow vectors
  • Showing motion direction
  • Annotating images with directional indicators
