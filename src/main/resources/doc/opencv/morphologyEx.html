MorphologyEx

Performs advanced morphological operations on an image. Combines basic operations (erode, dilate) to achieve more complex transformations.

Signature

Imgproc / cv2.morphologyEx(Mat src, Mat dst, int op, Mat kernel)

Parameters

  • Mat src - Source image; can have any number of channels.
  • Mat dst - Destination image of the same size and type as src.
  • int op - Type of morphological operation (see table below).
  • Mat kernel - Structuring element. Use getStructuringElement() to create.

Operation Types

OperationConstantFormulaUse Case
OpeningMORPH_OPENdilate(erode(src))Remove small bright spots (noise)
ClosingMORPH_CLOSEerode(dilate(src))Fill small holes in foreground
GradientMORPH_GRADIENTdilate(src) - erode(src)Find object boundaries/outlines
Top HatMORPH_TOPHATsrc - open(src)Extract bright spots on dark background
Black HatMORPH_BLACKHATclose(src) - srcExtract dark spots on bright background
Hit-or-MissMORPH_HITMISSerode(src, k1) & erode(~src, k2)Find specific patterns

Description

Opening (MORPH_OPEN):

  • Erosion followed by dilation
  • Removes small bright spots and thin protrusions
  • Preserves the overall shape and size of larger objects
  • Good for noise removal in binary images

Closing (MORPH_CLOSE):

  • Dilation followed by erosion
  • Fills small holes and gaps
  • Connects nearby objects
  • Good for filling gaps in detected objects

Gradient (MORPH_GRADIENT):

  • Difference between dilation and erosion
  • Produces the outline of objects
  • Useful for edge detection alternative

See Also

  • erode() - Basic erosion
  • dilate() - Basic dilation
