DFT - Discrete Fourier Transform

Performs a forward or inverse Discrete Fourier Transform on a 1D or 2D array. Used for frequency domain analysis and filtering.

Signature

Core / cv2.dft(Mat src, Mat dst, int flags)

Parameters

  • Mat src - Input array (real or complex).
  • Mat dst - Output array with complex values or real if DFT_REAL_OUTPUT.
  • int flags - DFT_INVERSE for inverse transform, DFT_SCALE to scale, etc.

Description

The Fourier Transform converts images from spatial domain to frequency domain, where:

  • Low frequencies: Slow changes, smooth regions
  • High frequencies: Rapid changes, edges, noise

Complete FFT workflow:

// Prepare image (optimal size for FFT)
int m = Core.getOptimalDFTSize(gray.rows());
int n = Core.getOptimalDFTSize(gray.cols());
Mat padded = new Mat();
Core.copyMakeBorder(gray, padded, 0, m - gray.rows(), 0, n - gray.cols(),
    Core.BORDER_CONSTANT, Scalar.all(0));

// Create complex image (real + imaginary)
padded.convertTo(padded, CvType.CV_32F);
List<Mat> planes = new ArrayList<>();
planes.add(padded);
planes.add(Mat.zeros(padded.size(), CvType.CV_32F));
Mat complexI = new Mat();
Core.merge(planes, complexI);

// Perform DFT
Core.dft(complexI, complexI);

// Shift zero-frequency to center
// (swap quadrants)
int cx = complexI.cols() / 2;
int cy = complexI.rows() / 2;
Mat q0 = complexI.submat(new Rect(0, 0, cx, cy));
Mat q1 = complexI.submat(new Rect(cx, 0, cx, cy));
Mat q2 = complexI.submat(new Rect(0, cy, cx, cy));
Mat q3 = complexI.submat(new Rect(cx, cy, cx, cy));
Mat tmp = new Mat();
q0.copyTo(tmp); q3.copyTo(q0); tmp.copyTo(q3);
q1.copyTo(tmp); q2.copyTo(q1); tmp.copyTo(q2);
