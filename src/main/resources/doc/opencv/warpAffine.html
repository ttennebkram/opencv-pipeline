warpAffine

Applies an affine transformation to an image. Affine transforms preserve parallel lines and include rotation, scaling, translation, and shearing.

Signature

Imgproc / cv2.warpAffine(Mat src, Mat dst, Mat M, Size dsize)

Parameters

  • Mat src - Input image.
  • Mat dst - Output image with specified size.
  • Mat M - 2x3 transformation matrix.
  • Size dsize - Size of the output image.

Description

The transformation is:

dst(x,y) = src(M00*x + M01*y + M02, M10*x + M11*y + M12)

Creating transformation matrices:

// Rotation around center
Point center = new Point(src.cols()/2, src.rows()/2);
double angle = 45;  // degrees
double scale = 1.0;
Mat M = Imgproc.getRotationMatrix2D(center, angle, scale);
Imgproc.warpAffine(src, dst, M, src.size());

// Translation
Mat M = new Mat(2, 3, CvType.CV_64F);
M.put(0, 0, 1, 0, tx,   // tx = horizontal shift
              0, 1, ty);  // ty = vertical shift
Imgproc.warpAffine(src, dst, M, src.size());

// From three point correspondences
MatOfPoint2f srcPts = new MatOfPoint2f(
    new Point(0, 0), new Point(100, 0), new Point(0, 100));
MatOfPoint2f dstPts = new MatOfPoint2f(
    new Point(10, 10), new Point(90, 20), new Point(20, 90));
Mat M = Imgproc.getAffineTransform(srcPts, dstPts);

Border modes:

  • BORDER_CONSTANT: Fill with specified color
  • BORDER_REPLICATE: Extend edge pixels
  • BORDER_REFLECT: Mirror at border
