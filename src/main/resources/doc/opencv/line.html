line

Draws a line segment connecting two points.

Signature

Imgproc / cv2.line(Mat img, Point pt1, Point pt2, Scalar color, int thickness)

Parameters

  • Mat img - Image where the line is drawn.
  • Point pt1 - First point of the line segment.
  • Point pt2 - Second point of the line segment.
  • Scalar color - Line color in BGR format.
  • int thickness - Line thickness in pixels.

Description

Examples:

// Draw a simple red line
Imgproc.line(img, new Point(10, 10), new Point(200, 100),
             new Scalar(0, 0, 255), 2);

// Draw with anti-aliasing (smoother)
Imgproc.line(img, new Point(10, 50), new Point(200, 150),
             new Scalar(0, 255, 0), 2, Imgproc.LINE_AA);

// Draw thick line
Imgproc.line(img, new Point(10, 100), new Point(200, 200),
             new Scalar(255, 0, 0), 5);

Line types:

  • LINE_4: 4-connected line (faster, more jagged)
  • LINE_8: 8-connected line (default)
  • LINE_AA: Anti-aliased line (smoother, slower)
