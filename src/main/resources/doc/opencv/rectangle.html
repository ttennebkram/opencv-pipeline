rectangle

Draws a rectangle outline or filled rectangle.

Signature

Imgproc / cv2.rectangle(Mat img, Point pt1, Point pt2, Scalar color, int thickness)

Imgproc / cv2.rectangle(Mat img, Rect rec, Scalar color, int thickness)

Parameters

  • Mat img - Image where the rectangle is drawn.
  • Point pt1 - Top-left corner of the rectangle.
  • Point pt2 - Bottom-right corner of the rectangle.
  • Rect rec - Rectangle to draw (alternative to two points).
  • Scalar color - Rectangle color in BGR format.
  • int thickness - Line thickness. Use -1 (FILLED) for filled rectangle.

Description

Examples:

// Draw rectangle using corner points
Imgproc.rectangle(img,
    new Point(50, 50),    // top-left
    new Point(200, 150),  // bottom-right
    new Scalar(0, 255, 0), 2);

// Draw rectangle using Rect
Rect roi = new Rect(250, 50, 100, 80);  // x, y, width, height
Imgproc.rectangle(img, roi, new Scalar(255, 0, 0), 2);

// Draw filled rectangle
Imgproc.rectangle(img,
    new Point(50, 200), new Point(150, 280),
    new Scalar(0, 0, 255), -1);  // -1 = filled

// Draw bounding box around detected object
Rect bbox = Imgproc.boundingRect(contour);
Imgproc.rectangle(img, bbox, new Scalar(0, 255, 255), 2);

Common use cases:

  • Drawing bounding boxes around detected objects
  • Marking regions of interest (ROI)
  • Visualizing face/object detection results
  • Creating simple UI elements
