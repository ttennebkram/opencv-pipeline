Median Blur

Blurs an image using a median filter. Excellent for removing salt-and-pepper noise while preserving edges.

Signature

Imgproc / cv2.medianBlur(Mat src, Mat dst, int ksize)

Parameters

  • Mat src - Input 1-, 3-, or 4-channel image. When ksize is 3 or 5, the image depth should be CV_8U, CV_16U, or CV_32F. For larger apertures, only CV_8U is supported.
  • Mat dst - Output image of the same size and type as src.
  • int ksize - Aperture linear size; it must be odd and greater than 1, for example: 3, 5, 7...

Description

The median filter replaces each pixel with the median value of all pixels in the neighborhood defined by the kernel size. Unlike averaging filters, the median filter uses the actual pixel values from the neighborhood.

Characteristics:

  • Excellent at removing salt-and-pepper (impulse) noise
  • Preserves edges better than linear filters
  • Output pixel is always an actual value from the input
  • Slower than box or Gaussian blur
  • Non-linear filter (cannot be separated)

How it works:

  1. For each pixel, collect all values in the kernel neighborhood
  2. Sort the collected values
  3. Replace the center pixel with the median (middle) value

  • bilateralFilter() - Edge-preserving blur
