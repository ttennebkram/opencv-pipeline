bitwise_xor

Calculates the per-element bit-wise exclusive OR (XOR) of two arrays or an array and a scalar.

Signature

Core / cv2.bitwise_xor(Mat src1, Mat src2, Mat dst, Mat mask)

Parameters

  • Mat src1 - First input array or scalar.
  • Mat src2 - Second input array or scalar.
  • Mat dst - Output array with the same size and type as the inputs.
  • Mat mask - Optional operation mask, 8-bit single channel (optional, can be empty Mat).

Description

Computes bitwise XOR for each element:

dst(x,y) = src1(x,y) XOR src2(x,y)    if mask(x,y) != 0

XOR returns 1 when inputs differ, 0 when they're the same.

Common use cases:

  • Difference detection: Highlight differences between two images
  • Toggle/flip pixels: XOR with white inverts, XOR with black preserves
  • Symmetric difference: Find regions in one mask but not both

Example - finding differences:

// Highlight differences between two frames
Mat diff = new Mat();
Core.bitwise_xor(frame1, frame2, diff);

// Or toggle specific pixels using a pattern
Mat pattern = new Mat(img.size(), CvType.CV_8UC1, new Scalar(255));
Core.bitwise_xor(img, pattern, inverted);  // Same as bitwise_not
