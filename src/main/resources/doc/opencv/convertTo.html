convertTo - Brightness/Contrast

Converts a Mat to another data type with optional scaling. Commonly used for brightness and contrast adjustment.

Signature

mat.convertTo(Mat dst, int rtype, double alpha, double beta)

Parameters

  • Mat dst - Output matrix.
  • int rtype - Desired output type (e.g., CvType.CV_8U). Use -1 for same as source.
  • double alpha - Scale factor (gain/contrast). Default 1.0.
  • double beta - Value added to scaled values (bias/brightness). Default 0.

Description

Each element is transformed by:

dst(x,y) = saturate(alpha * src(x,y) + beta)

Parameter effects:

  • alpha (gain): Controls contrast
    • alpha > 1: Increase contrast
    • alpha < 1: Decrease contrast
    • alpha = 1: No change
  • beta (bias): Controls brightness
    • beta > 0: Brighten image
    • beta < 0: Darken image
    • beta = 0: No change

Examples:

// Increase brightness by 50
src.convertTo(dst, -1, 1.0, 50);

// Increase contrast (alpha = 1.5)
src.convertTo(dst, -1, 1.5, 0);

// Increase both contrast and brightness
src.convertTo(dst, -1, 1.3, 30);

// Decrease brightness (darken)
src.convertTo(dst, -1, 1.0, -40);

// Convert to float type
src.convertTo(floatImg, CvType.CV_32F, 1.0/255.0, 0);

// Invert image (negative)
src.convertTo(dst, -1, -1, 255);
