Laplacian Operator

Calculates the Laplacian of an image. The Laplacian is a second-order derivative that highlights regions of rapid intensity change.

Signature

Imgproc / cv2.Laplacian(Mat src, Mat dst, int ddepth, int ksize)

Parameters

  • Mat src - Source image.
  • Mat dst - Destination image of the same size and same number of channels as src.
  • int ddepth - Desired depth of the destination image. Use CV_16S or CV_64F to avoid overflow.
  • int ksize - Aperture size used to compute the second-derivative filters. Must be positive and odd. Default is 1.

Description

The Laplacian operator calculates the sum of second-order x and y derivatives:

Laplacian(f) = ∂²f/∂x² + ∂²f/∂y²

This is approximated by convolving the image with a kernel. For ksize=1, a 3x3 kernel is used:

[ 0  1  0]
[ 1 -4  1]
[ 0  1  0]

Characteristics:

  • Detects edges in all directions simultaneously
  • Very sensitive to noise (usually requires pre-blurring)
  • Zero-crossings indicate edge locations
  • Produces double edges (one positive, one negative)

Typical workflow:

  1. Convert to grayscale
  2. Apply Gaussian blur to reduce noise
  3. Apply Laplacian
  4. Convert to absolute values for display

See Also

  • Scharr() - More accurate first derivative
  • Canny() - Multi-stage edge detector
