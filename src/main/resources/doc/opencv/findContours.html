findContours

Finds contours in a binary image. Contours are curves joining continuous points with the same intensity, useful for shape analysis and object detection.

Signature

Imgproc / cv2.findContours(Mat image, List<MatOfPoint> contours, Mat hierarchy, int mode, int method)

Parameters

  • Mat image - 8-bit single-channel binary image. Non-zero pixels are treated as 1's, zero as 0's.
  • List<MatOfPoint> contours - Detected contours. Each contour is a vector of points.
  • Mat hierarchy - Optional output containing topology information.
  • int mode - Contour retrieval mode (see table below).
  • int method - Contour approximation method.

Retrieval Modes

ModeDescription
RETR_EXTERNALOnly outermost contours
RETR_LISTAll contours without hierarchy
RETR_TREEFull hierarchy of nested contours
RETR_CCOMPTwo-level hierarchy (external and holes)

Approximation Methods

MethodDescription
CHAIN_APPROX_NONEStore all contour points
CHAIN_APPROX_SIMPLECompress horizontal, vertical, diagonal segments
CHAIN_APPROX_TC89_L1Teh-Chin chain approximation

Common Operations

// Find and draw contours
List<MatOfPoint> contours = new ArrayList<>();
Mat hierarchy = new Mat();
Imgproc.findContours(binary, contours, hierarchy,
                     Imgproc.RETR_EXTERNAL, Imgproc.CHAIN_APPROX_SIMPLE);

// Draw all contours
Imgproc.drawContours(dst, contours, -1, new Scalar(0, 255, 0), 2);

// Contour properties
double area = Imgproc.contourArea(contours.get(0));
double perimeter = Imgproc.arcLength(new MatOfPoint2f(contours.get(0).toArray()), true);
Rect bbox = Imgproc.boundingRect(contours.get(0));
