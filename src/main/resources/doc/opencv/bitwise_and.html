bitwise_and

Calculates the per-element bit-wise conjunction (AND) of two arrays or an array and a scalar.

Signature

Core / cv2.bitwise_and(Mat src1, Mat src2, Mat dst, Mat mask)

Parameters

  • Mat src1 - First input array or scalar.
  • Mat src2 - Second input array or scalar.
  • Mat dst - Output array with the same size and type as the inputs.
  • Mat mask - Optional operation mask, 8-bit single channel (optional, can be empty Mat).

Description

Computes bitwise AND for each element:

dst(x,y) = src1(x,y) AND src2(x,y)    if mask(x,y) != 0
         = 0                            otherwise

Common use cases:

  • Masking: Extract regions from an image using a binary mask
  • Combining masks: Find intersection of two masks
  • ROI extraction: Keep only pixels where mask is white

Example - applying a mask:

// Keep only the parts of image where mask is white
Mat result = new Mat();
Core.bitwise_and(image, image, result, mask);

// Or combine two images using masks
Core.bitwise_and(img1, mask1, result1);
Core.bitwise_and(img2, mask2, result2);
Core.add(result1, result2, combined);
