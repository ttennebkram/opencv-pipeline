HoughCircles

Finds circles in a grayscale image using the Hough Gradient method.

Signature

Imgproc / cv2.HoughCircles(Mat image, Mat circles, int method, double dp, double minDist, double param1, double param2, int minRadius, int maxRadius)

Parameters

  • Mat image - 8-bit, single-channel grayscale input image.
  • Mat circles - Output vector of circles. Each circle is (x, y, radius).
  • int method - Detection method. Use HOUGH_GRADIENT.
  • double dp - Inverse ratio of accumulator resolution (1 = same as image, 2 = half resolution).
  • double minDist - Minimum distance between circle centers.
  • double param1 - Upper Canny edge threshold (lower is param1/2).
  • double param2 - Accumulator threshold for circle centers. Smaller = more circles detected.
  • int minRadius - Minimum circle radius (0 for no minimum).
  • int maxRadius - Maximum circle radius (0 for no maximum).

Description

Uses the Hough Gradient method, which internally applies Canny edge detection and then finds circle centers using gradient information.

Tip: Apply GaussianBlur before HoughCircles to reduce noise and improve detection accuracy.

Typical parameter values:

  • dp = 1: Full resolution accumulator
  • minDist = image height / 8: Prevent overlapping circles
  • param1 = 100: Canny high threshold
  • param2 = 30-50: Lower = more circles (may include false positives)

Drawing detected circles:

for (int i = 0; i < circles.cols(); i++) {
    double[] c = circles.get(0, i);
    Point center = new Point(c[0], c[1]);
    int radius = (int) Math.round(c[2]);
    // Draw center
    Imgproc.circle(dst, center, 3, new Scalar(0, 255, 0), -1);
    // Draw outline
    Imgproc.circle(dst, center, radius, new Scalar(0, 0, 255), 2);
}
