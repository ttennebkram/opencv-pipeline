Point

The Point class represents a 2D point with x and y coordinates.

Constructors

new Point()                   // Default: x=0, y=0
new Point(double x, double y)
new Point(double[] vals)      // From array [x, y]

Properties

PropertyTypeDescription
xdoubleThe x-coordinate (horizontal position)
ydoubleThe y-coordinate (vertical position)

Methods

MethodDescription
dot(Point p)Computes dot product with another point
inside(Rect r)Checks if point is inside a rectangle
clone()Creates a copy of this Point

Common Usage

Creating Points

// A point at coordinates (100, 50)
Point pt = new Point(100, 50);

// Origin
Point origin = new Point(0, 0);

Drawing Operations

// Draw a line from (10,10) to (200,200)
Imgproc.line(img, new Point(10, 10), new Point(200, 200),
             new Scalar(0, 255, 0), 2);

// Draw a circle at center (150, 150)
Imgproc.circle(img, new Point(150, 150), 50,
               new Scalar(255, 0, 0), -1);

Template Matching

// Find best match location
Core.MinMaxLocResult result = Core.minMaxLoc(matchResult);
Point matchLoc = result.maxLoc;  // Location of best match

Coordinate System

OpenCV uses a coordinate system where:

  • x increases to the right (column index)
  • y increases downward (row index)
  • Origin (0,0) is at the top-left corner
    (0,0) -----> x
      |
      |
      v
      y
