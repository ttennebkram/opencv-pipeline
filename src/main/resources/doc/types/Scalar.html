Scalar

The Scalar class represents a 4-element vector, commonly used for pixel values and colors.

Constructors

new Scalar(double v0)                           // Single value (grayscale)
new Scalar(double v0, double v1, double v2)     // BGR color
new Scalar(double v0, double v1, double v2, double v3)  // BGRA with alpha
new Scalar(double[] vals)                       // From array

Properties

PropertyDescription
valArray of 4 double values [v0, v1, v2, v3]
Important: OpenCV uses BGR order, not RGB!
Scalar(Blue, Green, Red) - e.g., Scalar(255, 0, 0) is blue, not red.

Common Usage

Drawing with Colors

// Draw a red circle (BGR: Blue=0, Green=0, Red=255)
Imgproc.circle(img, center, radius, new Scalar(0, 0, 255), 2);

// Draw a filled green rectangle
Imgproc.rectangle(img, pt1, pt2, new Scalar(0, 255, 0), -1);

Creating Solid Color Images

// Create a 640x480 blue image
Mat blueImg = new Mat(480, 640, CvType.CV_8UC3, new Scalar(255, 0, 0));

Color Range for inRange()

// Detect blue objects in HSV color space
Scalar lowerBlue = new Scalar(100, 50, 50);
Scalar upperBlue = new Scalar(130, 255, 255);
Core.inRange(hsv, lowerBlue, upperBlue, mask);
