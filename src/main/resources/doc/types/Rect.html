Rect

The Rect class represents a rectangle defined by its top-left corner position and size.

Constructors

new Rect()                                // Default: all zeros
new Rect(int x, int y, int width, int height)
new Rect(Point p, Size s)                 // From point and size
new Rect(Point p1, Point p2)              // From two corners
new Rect(double[] vals)                   // From array [x, y, width, height]

Properties

PropertyTypeDescription
xintX-coordinate of top-left corner
yintY-coordinate of top-left corner
widthintWidth of rectangle
heightintHeight of rectangle

Methods

MethodDescription
tl()Returns top-left corner as Point
br()Returns bottom-right corner as Point
size()Returns Size(width, height)
area()Returns width * height
empty()Returns true if width <= 0 or height <= 0
contains(Point p)Checks if point is inside rectangle
clone()Creates a copy

Common Usage

Creating Rectangles

// Rectangle at (100,50) with size 200x150
Rect roi = new Rect(100, 50, 200, 150);

// From two corner points
Rect rect = new Rect(new Point(10, 10), new Point(100, 80));

Cropping (Region of Interest)

// Extract a region from an image
Rect roi = new Rect(50, 50, 100, 100);
Mat cropped = new Mat(img, roi);  // Shares data with original

// To get a copy:
Mat croppedCopy = img.submat(roi).clone();

Drawing Rectangles

// Draw rectangle outline
Imgproc.rectangle(img, new Rect(10, 10, 100, 80),
                  new Scalar(0, 255, 0), 2);

// Or using corner points
Imgproc.rectangle(img, new Point(10, 10), new Point(110, 90),
                  new Scalar(0, 255, 0), 2);

Bounding Rectangles

// Get bounding rectangle of a contour
Rect boundingRect = Imgproc.boundingRect(contour);

// Get minimum area rotated rectangle
RotatedRect minRect = Imgproc.minAreaRect(contour2f);
