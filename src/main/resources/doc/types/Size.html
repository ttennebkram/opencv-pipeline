Size

The Size class represents the size of a 2D object, specified by its width and height.

Constructors

new Size()                    // Default: width=0, height=0
new Size(double width, double height)
new Size(double[] vals)       // From array [width, height]

Properties

PropertyTypeDescription
widthdoubleThe width dimension
heightdoubleThe height dimension

Methods

MethodDescription
area()Returns width * height
empty()Returns true if width <= 0 or height <= 0
clone()Creates a copy of this Size

Common Usage

Creating a Size

// For a 640x480 image
Size size = new Size(640, 480);

// Kernel size for blur operations (must be odd)
Size kernelSize = new Size(5, 5);

With Image Operations

// Resize an image
Imgproc.resize(src, dst, new Size(320, 240));

// Gaussian blur with 5x5 kernel
Imgproc.GaussianBlur(src, dst, new Size(5, 5), 0);

Notes

  • For kernel sizes (blur, morphology), both width and height must be positive odd numbers (1, 3, 5, 7, ...)
  • Size uses double values, but many operations expect integer sizes
  • A Size of (0, 0) often means "compute automatically from other parameters"
