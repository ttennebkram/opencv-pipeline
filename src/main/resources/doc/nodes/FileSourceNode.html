File Source Node

Loads images or video from files on disk. This is typically the starting point for file-based pipelines.

Supported Formats

Images

  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • BMP (.bmp)
  • TIFF (.tif, .tiff)
  • WebP (.webp)

Video

  • MP4 (.mp4)
  • AVI (.avi)
  • MOV (.mov)
  • Other formats supported by OpenCV's VideoCapture

Properties

  • File Path - Absolute or relative path to the image/video file
  • Loop - For videos, restart from beginning when finished
Tip: Use relative paths (relative to the pipeline file location) to make pipelines portable between machines.

Behavior

  • Images: Outputs the same frame repeatedly
  • Videos: Outputs frames sequentially at the capture rate
Note: If the file cannot be found or read, an error message is displayed in the node's status area.
