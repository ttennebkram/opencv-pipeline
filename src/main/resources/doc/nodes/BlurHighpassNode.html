Blur Highpass Node

A spatial-domain highpass filter that extracts high-frequency details (edges, textures) by subtracting a blurred version of the image from the original.

Algorithm

output = |input - GaussianBlur(input)|

The absolute difference ensures positive values in the output.

Properties

  • Kernel Size - Size of the Gaussian blur kernel (larger = more detail extracted)

How It Works

Low-pass filtering (blur) removes high frequencies. By subtracting the blurred image from the original, we isolate what the blur removed: the high-frequency content.

Applications

  • Edge enhancement
  • Texture analysis
  • Feature extraction
  • Image sharpening (when added back to original)
Tip: A larger kernel size extracts coarser details. A smaller kernel size extracts finer details and edges.
Note: This is a simple spatial highpass filter. For more precise frequency control, use the FFT-based highpass filters.
