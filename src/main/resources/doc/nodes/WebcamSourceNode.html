Webcam Source Node

Captures live video frames from a connected camera or webcam.

Properties

  • Camera Index - Which camera to use (0 = default/first camera, 1 = second camera, etc.)

How It Works

The Webcam Source uses OpenCV's VideoCapture class to grab frames from the camera. Frames are captured at the camera's native frame rate and resolution.

Tip: If you have multiple cameras, try different index values (0, 1, 2...) to find the one you want.

Troubleshooting

  • No video: Check that the camera is connected and not in use by another application
  • Wrong camera: Try a different camera index
  • Permission denied: On macOS, grant camera permission in System Preferences
Note: Camera permissions may be required on macOS and some Linux systems. The application may need to be restarted after granting permissions.
