Clone Node

Duplicates the input frame to multiple outputs, allowing parallel processing branches in your pipeline.

Outputs

The Clone node has two output connections:

  • Output 1 (top) - First copy of the input frame
  • Output 2 (bottom) - Second copy of the input frame

Use Cases

  • Apply different filters to the same image for comparison
  • Create parallel processing branches that merge later
  • Send one copy to display while processing another
Tip: Each output receives an independent clone of the input Mat, so downstream nodes can modify them without affecting each other.
Note: When disabled, the Clone node passes input to Output 1 only. Output 2 receives nothing.
