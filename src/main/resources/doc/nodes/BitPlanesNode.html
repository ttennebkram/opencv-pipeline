Bit Planes Node

Decomposes an image into its bit planes, revealing the contribution of each bit position to the overall image.

Variants

  • Bit Planes (Grayscale) - Works on single-channel grayscale images
  • Bit Planes (Color) - Processes each color channel independently

How It Works

Each pixel in an 8-bit image has values from 0-255, which can be represented as 8 binary bits. This node extracts individual bit planes by masking each bit position:

  • Bit 7 (MSB) - Most significant bit, contains most visual information
  • Bit 6-4 - High-order bits, significant image detail
  • Bit 3-1 - Low-order bits, fine detail and noise
  • Bit 0 (LSB) - Least significant bit, mostly noise

Properties

  • Bit - Which bit plane to extract (0-7)
  • Gain - Multiplier to make the result visible (since single bits are 0 or 1)
Tip: Higher bit planes (7, 6, 5) contain the main image structure. Lower bit planes (0, 1, 2) often reveal hidden patterns, watermarks, or noise.

Applications

  • Image analysis and forensics
  • Steganography detection
  • Understanding image compression artifacts
  • Educational visualization of digital images
Note: The output is scaled by the gain factor. A gain of 255 makes the binary plane fully visible (0 or 255 output values).
