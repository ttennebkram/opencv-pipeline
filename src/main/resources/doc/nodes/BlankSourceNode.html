Blank Source Node

Generates frames filled with a solid color. Useful for testing, creating backgrounds, or as a canvas for drawing operations.

Properties

  • Width - Frame width in pixels
  • Height - Frame height in pixels
  • Color - Fill color (click to open color picker)

Use Cases

  • Creating a background for overlay composition
  • Testing drawing nodes (Circle, Rectangle, Text, etc.)
  • Generating masks with specific colors
  • Creating a canvas for procedural graphics
Tip: Use a black blank source with drawing nodes to create custom masks or test patterns.
Note: Blank Source generates BGR color images (3 channels). For grayscale, follow with a conversion node.
